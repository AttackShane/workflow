# 01 · 项目概览

## 1.1 项目定位

**Coze 工作流编辑器 & 转换器** 是一个基于浏览器原生 ES Module 的纯前端工作流工具，旨在为 Coze 平台用户提供：

1. **工作流可视化编辑**：拖拽式画布，所见即所得地搭建工作流。
2. **YAML ↔ Coze 剪贴板格式双向转换**：打通与 Coze 官方生态的数据交换通道。
3. **本地工作流管理**：无需后端即可在浏览器中存储、复用、整理多份工作流。
4. **离线可用**：构建产物是单文件 HTML，可通过 `file://` 协议直接打开。

## 1.2 核心能力矩阵

| 能力 | 页面 | 主要模块 |
| --- | --- | --- |
| 工作流管理（CRUD、搜索、批量、排序、模板） | 工作流管理页 | `manager.js` / `manager-templates.js` |
| YAML → Coze 剪贴板格式 | 转换器页 | `converter.js` / `converter-renderer.js` / `converter-keyboard.js` |
| Coze 剪贴板格式 → YAML | 转换器页 | `converter-reverse.js` |
| 可视化编辑（节点/边/选择/对齐/历史/撤销重做） | 编辑器页 | `editor-ui.js` / `editor-core.js` / `editor-canvas.js` / `editor-node.js` |
| 剪贴板双向（编辑器内 Ctrl+C/V） | 编辑器页 | `editor-clipboard.js` / `editor-clipboard-paste.js` |
| 节点搜索、定位、小地图、自动保存、触摸支持 | 编辑器页 | `editor-search.js` / `editor-canvas.js` / `editor-autosave.js` |
| 国际化（中英）、主题（深浅）、模态对话框 | 全局 | `shared-i18n.js` / `shared-theme.js` / `shared-dialog.js` |

## 1.3 目标用户

- **Coze 工作流设计者**：在本地复用、修改、备份工作流定义。
- **AI 应用开发者**：通过 YAML 形式版本化管理工作流。
- **教学/演示场景**：直观展示工作流拓扑。

## 1.5 关键指标

- **50 种节点类型**（start / end / llm / condition / loop / batch / question / http / database / image_generate / video_generation / ...）。
- **25 个测试套件 / 1127 测试用例**（jest）。
- **零外部运行依赖**：生产构建产物为单文件 HTML，仅需 `file://` 即可运行。
- **纯 ES6 模块**：无 React / Vue / jQuery。
- **主题/语言自适应**：CSS 变量 + i18n 监听器自动响应。
